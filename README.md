# 褪せ人の語彙録 — JLPT N2 Vocab SRS

> *The Tarnished's Lexicon · SRS Edition*

エルデンリング風のダーク・ファンタジーUIで学ぶ、JLPT N2 単語学習アプリです。
SM-2アルゴリズムによる間隔反復（SRS）で効率的な語彙習得をサポートします。

---

## 機能

- **3つの学習モード**
  - `日→英` — 日本語を見て英語の意味を答える
  - `英→日` — 英語を見て日本語を答える
  - `読み方` — 漢字の読み方（ひらがな）を答える

- **SM-2 間隔反復 (SRS)**
  - 「また忘れた / 難しかった / 覚えていた / 簡単だった」の4段階評価
  - 評価に応じて次の復習タイミングを自動スケジュール
  - 復習期限を過ぎた単語を優先出題

- **10問ラウンド制**
  - 1ラウンド10問の4択クイズ
  - ラウンド終了時にスコアとフィードバックを表示

- **学習履歴**
  - 最大2,000件の解答履歴をブラウザに保存
  - 正解 / 不正解でフィルタリング可能
  - ページネーション付き一覧表示

- **統計トラッキング**
  - 累計正答数・正解率・連鎖（ストリーク）・総問数を常時表示

- **キーボードショートカット**
  - `1`〜`4` — 選択肢を選ぶ
  - `Space` — 答えを確認
  - `Enter` — 次の問題へ

## 使い方

ローカルで起動する場合は、CORSの制約を避けるためライブサーバーを使うことを推奨します。

```bash
# VS Code の Live Server 拡張、または npx serve を使う例
npx serve .
```

ブラウザで `index.html` を直接開くだけでも動作します。

---

## データの保存先

学習データはすべて **ブラウザの `localStorage`** に保存されます。

| キー | 内容 |
|------|------|
| `quiz-history` | 解答履歴（最大2,000件） |
| `global-stats` | 累計統計情報 |
| `srs-cards` | 各単語のSRSカード状態 |

ブラウザのデータを削除すると学習進捗もリセットされます。
